# Инструкции по использованию модели классификации изображений

## Описание

Этот проект представляет собой пример решения задачи классификации изображений с использованием глубокого обучения. 
Модель создана с использованием библиотеки PyTorch и предназначена для определения классов тканей на изображениях.

### Данные

Модель была обучена на наборах train_tiny и train. Тестироание произведено на test_tiny и test соотв.

### Архитектура модели

Модель основана на архитектуре ResNet18 из библиотеки torchvision. Лучшая модель сохранена в best.pth

### Результаты обучения

Модель достигла точности % на тестовом наборе данных, сбалансированная точность составила %.
